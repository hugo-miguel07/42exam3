# 42exam3

## Description

**42exam3** is a personal study and training repository dedicated to preparing for **Exam 03** of the **42 School** curriculum.

Exam 03 typically requires students to implement one of the following subjects from scratch under timed conditions:

- **ft_printf** – a partial re-implementation of the C standard `printf` function
- **get_next_line** – a function that reads a single line from a file descriptor

The goal of this repository is to practice these subjects repeatedly until they can be written confidently and correctly within the exam time limit, reinforcing a solid understanding of variadic functions, file descriptors, static variables, and memory management in C.

## How to Use

Clone this repository and navigate into the subject you want to practice:

```sh
git clone https://github.com/hugo-miguel07/42exam3.git
cd 42exam3/ft_printf
```

Compile and test your implementation manually:

```sh
gcc -Wall -Wextra -Werror *.c -o test && ./test
```

Repeat the exercise as many times as needed until you can write the solution from memory within the exam constraints.
